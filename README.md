Samsung Innovation Campus

This folder contains practice materials from the SIC sessions, including datasets and notebooks.

Contents
- datasets/: CSV and text datasets used in exercises.
- notebooks/: Jupyter notebooks for lessons and practice.
- media/: Supporting markdown or assets.

Notes
- Start with notebooks in chronological order if you are following along with sessions.
- Datasets are referenced by relative paths inside the notebooks.

Syllabus and lesson types (from notebooks)
- Hands-on coding exercises and recap sessions (coding_exercise, exercise, ex36, ex37, ex_0508, ex_506).
- Text processing and regular expressions (ex_0501-RegularExpressions).
- Classical ML models: logistic regression (logistic_regression).
- Decision trees: basic concepts and separate classification/regression notebooks
	(decisiontree, decisiontree_classifier, decisiontree_regressor).
- Single-layer perceptron demo with logic gate (slp_AND).
- Day-wise session notebooks (day3, newDay).
